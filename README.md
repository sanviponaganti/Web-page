# 🌍 Trekko – Travel & Tourism Website

Welcome to **Trekko** — a modern, responsive travel and tourism landing page designed to inspire travelers to explore breathtaking destinations across India. Built using **HTML5, CSS3, and JavaScript**, the website delivers a clean, engaging, and user-friendly experience while showcasing curated destinations, featured travel packages, blogs, and essential travel information.

---

## 🚀 Live Demo

Experience the live website here:

👉 **https://sanviponaganti.github.io/Web-page/**

---

## ✨ Features

- 📱 **Fully Responsive Design** – Optimized for mobile phones, tablets, laptops, and desktop screens.
- 🍔 **Responsive Navigation Menu** – Interactive mobile navigation with a smooth hamburger menu.
- 🏔️ **Engaging Hero Section** – Eye-catching landing section with clear call-to-action buttons.
- 🌍 **Popular Destinations Showcase** – Explore featured travel destinations with elegant hover effects.
- 🎒 **Featured Tour Packages** – Display of curated travel packages with pricing, ratings, and trip duration.
- 📰 **Travel Blog Section** – Discover travel stories, inspiration, and destination highlights.
- 📩 **Newsletter Subscription** – Stay updated with the latest travel offers and news.
- ⬆️ **Scroll-to-Top Button** – Smooth navigation for enhanced user experience.

---

## 🛠️ Tech Stack

- **HTML5** – Semantic and accessible webpage structure.
- **CSS3** – Responsive styling using CSS Variables, Flexbox, Grid, and Media Queries.
- **JavaScript (ES6)** – DOM manipulation, event handling, and interactive UI functionality.
- **Ionicons** – Modern icon library.
- **Google Fonts** – Elegant typography for an enhanced visual experience.
